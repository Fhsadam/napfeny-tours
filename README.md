# Napfény Tours

Web-programozás 1 beadandó projekt.

A webalkalmazás az Utazás – 3 táblás mintaadatbázisra épül. Az oldalon elérhető főoldal, képgaléria, kapcsolat űrlap, üzenetkezelés, valamint egy admin jogosultsághoz kötött CRUD felület a szállodák adatainak kezelésére.

## Fő funkciók
- regisztráció, bejelentkezés, kilépés
- képgaléria és képfeltöltés
- kapcsolat űrlap kliens- és szerveroldali ellenőrzéssel
- üzenetek listázása
- admin CRUD a `szalloda` táblához
- reszponzív megjelenés
